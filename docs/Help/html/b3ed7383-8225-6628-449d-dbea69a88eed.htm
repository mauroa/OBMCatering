Actualizar
Actualiza los datos de una determinada factura Solo se permite actualizar el estado de la factura, es decir si fue cobrada o no
Crear
Crea una nueva factura en el sistema, lo que implica que una determinada orden de venta ha sido aprobada Por cada orden de venta o pedido debera crearse una factura que la respalde

Obtiene el listado completo de facturas del sistema
Obtener(Boolean)
Obtiene el listado de facturas segun si fueron cobradas o no
Obtener(Int32)
Obtiene una factura determinada segun su identificador
Obtener(Cliente)
Obtiene el listado de facturas de determinado cliente
Obtener(OrdenVenta)
