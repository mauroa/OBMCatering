CrearPasswordHash
Crea un hash para un password determinado Como medida de seguridad no se guardan passwords en texto claro en la base de datos, con lo cual se utiliza una funcion hash conocida (MD5) que convierte al texto en otro texto con longitud fija y cuyo valor sera unico por cada valor de texto original. Esto segura que cada hash sera unico para cada valor de texto, con lo cual calculando el hash de un texto y comparandolo con el hash guardado en la base de datos sera suficiente para asegurar que son el mismo valor y sera prueba suficiente para autenticar al usuario
